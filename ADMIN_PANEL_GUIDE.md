## 🎯 Все 7 Экранов Админ-Панели

### **1️⃣ GENERATE (Генерация)**
**URL:** `/admin/generate`

Здесь ты запускаешь генерацию страниц.

**Что делать:**
```
1. Выбери режим:
   - Режим 1: Генерация по городам + услуги
   - Режим 2: Генерация по ключевым словам

2. Заполни форму:
   - Режим 1: выбери нишу, загрузи города, выбери услуги
   - Режим 2: введи ключевое слово, выбери тип контента, укажи количество

3. Нажми "🚀 Запустить Генерацию"

4. Получишь Batch ID (сохрани его!)
```

**Пример:**
```
Mode 1:
  Ниша: Сантехнические услуги
  Города: almaty, turgen, shymkent (300 городов)
  Услуги: remont-trub, ustranenie-zasora
  
  → Результат: 300 × 2 = 600 страниц

Mode 2:
  Ключевое слово: "как отремонтировать трубу"
  Тип: Blog
  Количество: 100
  
  → Результат: 100 информационных статей
```

---

### **2️⃣ MONITOR (Мониторинг)**
**URL:** `/admin/monitor`

Следишь за прогрессом генерации в реальном времени.

**Что делать:**
```
1. Скопируй Batch ID из сообщения после запуска
2. Вставь в поле "Batch ID"
3. Нажми "🔍 Проверить"
4. Смотри:
   - Прогресс (%)
   - Статистику (готово, ошибок, уникальные)
   - Логи (что происходит прямо сейчас)
```

**Что видишь:**
```
✅ Готово: 204 страниц
⏳ В процессе: 96 страниц
❌ Ошибок: 0
✅ Уникальные: 195 (95%)
⚠️ Дубликаты: 9 (на переделку)
⏱️ Осталось ~15 минут
```

---

### **3️⃣ NICHES (Управление Нишами)**
**URL:** `/admin/niches`

Управляешь нишами (услугами).

**Что делать:**
```
1. Просматривай список своих ниш
2. Нажми "+ Добавить Нишу" для создания новой
3. Или редактируй/удаляй существующие

Примеры ниш:
  - Сантехнические услуги
  - Услуги слесаря
  - Электрические услуги
```

**Каждая ниша содержит:**
- Название (slug)
- Список услуг (5-10 услуг в каждой)
- Локальные факты (водоканал, цены и т.д.)
- Шаблон HTML

---

### **4️⃣ TEMPLATES (Управление Шаблонами)**
**URL:** `/admin/templates`

Управляешь HTML шаблонами дизайна.

**Что делать:**
```
1. Нажми "+ Загрузить Шаблон"
2. Выбери HTML файл с дизайном
3. Укажи для какой ниши этот шаблон
4. Готово! Теперь платформа будет использовать этот дизайн
```

**Требования к шаблону:**
- HTML файл с дизайном страницы
- Специальные маркеры: `data-pseo-block="..."`
- Плейсхолдеры: `{{hero.title}}`, `{{sections.0.content}}`

Смотри **TEMPLATE_GUIDE.md** для полного руководства.

---

### **5️⃣ LOCATIONS (Управление Городами)**
**URL:** `/admin/locations`

Управляешь городами и локальными данными.

**Что делать:**
```
1. Нажми "+ Загрузить Города (CSV)"
2. Выбери CSV файл с городами
3. Формат CSV:
   city,region,population,waterUtility,avgPrice
   almaty,Almaty Region,1400000,АО АлматыСу,150
   turgen,Zhambyl Region,45000,Водоканал,80

4. Загружай. Платформа автоматически добавит города.
```

**Что нужно в CSV:**
- city (название города)
- region (область)
- population (население)
- waterUtility (коммунальная служба)
- avgPrice (средняя цена услуги)

---

### **6️⃣ QA (Проверка Качества)**
**URL:** `/admin/qa`

Проверяешь качество страниц вручную (только 5% требует проверки!).

**Что делать:**
```
1. Просматриваешь текст страницы
2. Проверяешь скор уникальности (должен быть > 40%)
3. Нажимаешь:
   ✅ Одобрить → Страница готова к публикации
   ❌ Отклонить → Страница переделается автоматически
   ✏️ Редактировать → Редактируешь текст вручную
```

**Статистика:**
```
⚠️ Требует проверки: 5 страниц
✅ Одобрено: 990 страниц
❌ Отклонено: 5 страниц
```

---

### **7️⃣ PUBLISH (Волновая Публикация)**
**URL:** `/admin/publish`

Публикуешь страницы волнами для безопасной индексации.

**Как работает:**
```
Волна 1: Публикуем 100 страниц
         ↓ (ждём 3 дня)
         Проверяем индексацию в GSC
         
Волна 2: Публикуем следующие 100 страниц
         ↓ (ждём 3 дня)
         
Волна 3, 4, 5... (процесс повторяется)
```

**Результаты Волны 1:**
```
📊 Статус: ✅ ОПУБЛИКОВАНА
📈 Проиндексировано: 98/100 (98%)
📊 Impressions: 234
👆 Clicks: 12
```

**Что делать:**
```
1. Смотри статус всех волн
2. Нажми "🚀 Опубликовать Сейчас" чтобы запустить следующую волну
3. Платформа автоматически создаёт следующие волны
```

## 🎯 Пошаговый Процесс Работы

### Первый запуск (до первой генерации)

```
1. Перейди в "Ниши"
   → Создай нишу (например, Сантехнические услуги)

2. Перейди в "Шаблоны"
   → Загрузи HTML дизайн страницы
   → Укажи для какой ниши

3. Перейди в "Города"
   → Загрузи CSV с городами
   → 100-1000 городов

Готово к генерации!
```

### Запуск генерации

```
1. Перейди в "Генерация"

2. Выбери режим:
   - Режим 1: города + услуги
   - Режим 2: ключевое слово

3. Заполни форму

4. Нажми "🚀 Запустить Генерацию"

5. Скопируй Batch ID из ответа

Генерация началась!
```

### Мониторинг

```
1. Перейди в "Мониторинг"

2. Вставь Batch ID

3. Нажми "🔍 Проверить"

4. Смотри прогресс:
   - Каждые 3 секунды обновляется
   - Видишь статус, логи, статистику

Ждёшь пока завершится...
```

### Проверка качества

```
1. Перейди в "QA"

2. Просматриваешь страницы (только 5%)

3. Одобряешь или отклоняешь

4. Готово!
```

### Публикация

```
1. Перейди в "Публикация"

2. Нажми "🚀 Опубликовать Сейчас"

3. Волна 1 публикуется
   ↓ (ждёшь 3 дня)
   
4. Нажимаешь снова для Волны 2

5. Процесс повторяется...
```

## 💡 Советы

### Совет 1: Batch ID
```
Всегда сохраняй Batch ID когда запускаешь генерацию.
Он нужен для проверки статуса в Monitor.
```

### Совет 2: CSV Файл
```
Создавай CSV в Excel/Google Sheets:
- Колонки: city, region, population, waterUtility, avgPrice
- Сохраняй как CSV UTF-8
- Проверь что нет пустых строк
```

### Совет 3: Шаблон
```
Один HTML шаблон используется для всех городов.
Платформа автоматически вставляет данные города.
Например: {{localFacts.waterUtility}} → "АО АлматыСу"
```

### Совет 4: QA Проверка
```
Обычно требует проверки только 5% страниц.
95% проходят автоматическую проверку качества.
Быстро одобряй - процесс простой!
```

### Совет 5: Волны
```
Не публикуй всё сразу!
Волны - безопаснее для SEO.
Google не понизит домен если публикуешь волнами.
```

## 🚨 Проблемы и Решения

### "❌ Ошибка при запуске генерации"
```
Проверь:
1. Выбрана ли ниша?
2. Загружены ли города?
3. Выбраны ли услуги?
4. Все ли поля заполнены?
```

### "Мониторинг не обновляется"
```
1. Проверь правильность Batch ID
2. Скопируй ещё раз и вставь
3. Нажми "🔍 Проверить" снова
```

### "Волна не публикуется"
```
Проверь:
1. Есть ли одобренные страницы?
2. Может быть все уже опубликованы?
3. Обнови страницу (F5)
```
