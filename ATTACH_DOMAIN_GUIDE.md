# 🔗 Как Подвязать Страницы к Домену

## 📌 Самое Простое Объяснение

**Представь что:**
- У тебя **1000 страниц** с контентом ✅
- У тебя **домен santehnik.kz** ✅
- Теперь нужно их **связать вместе** 🔗

**Вот как это работает в 3 клика:**

```
1. Админ-панель → "Подвязать Домен"
2. Выбираешь 1000 страниц (или все сразу)
3. Пишешь: santehnik.kz
4. Нажимаешь: "ПОДВЯЗАТЬ 1000 СТРАНИЦ"
5. ГОТОВО! Все на домене! 🎉
```

---

## 🎯 Пошаговая Инструкция

### ШАГ 1: Открыть Раздел

```
1. Открыть админ-панель: http://localhost:3000/admin
2. Нажать в меню: "Подвязать Домен" 🔗
3. Откроется окно с двумя колонками
```

### ШАГ 2: Выбрать Страницы

**Левая колонка - Сгенерированные Страницы:**

```
☑ Выбрать все (1000 страниц)
├─ ☑ Ремонт труб в Алматы        (almaty-remont-trub)
├─ ☑ Ремонт труб в Тургене       (turgen-remont-trub)
├─ ☑ Ремонт труб в Шымкенте      (shymkent-remont-trub)
├─ ☑ Устранение засора в Алматы  (almaty-ustranenie-zasora)
└─ [еще 996 страниц...]
```

**Опции:**
- Выбрать всё сразу: нажать чекбокс "Выбрать все (1000)"
- Выбрать частично: нажать чекбоксы по одному
- Отменить выделение: нажать чекбокс ещё раз

### ШАГ 3: Ввести Домен

**Правая колонка - Настройка Домена:**

```
📧 Введи домен
┌────────────────────────────┐
│ santehnik.kz               │  ← Вводишь домен
└────────────────────────────┘

Примеры:
✅ santehnik.kz
✅ elektrik.kz
✅ remont-almaty.kz
✅ your-domain.com
```

### ШАГ 4: Подвязать

```
Статус:
┌─────────────────┐
│ Выбрано: 1000   │
│ Доступно: 1000  │
└─────────────────┘

Нажать кнопку:
[🔗 ПОДВЯЗАТЬ 1000 СТРАНИЦ]

Ждём... (обычно секунда)

Результат:
✅ 1000 страниц успешно подвязаны к домену santehnik.kz!
```

├────┼──────────────────┼──────────────────┼──────────────────┤

## 🌍 Как Домен Показывает Страницы

### Когда Пользователь Идёт на santehnik.kz/almaty-remont-trub:

```
1. Браузер открывает: santehnik.kz/almaty-remont-trub

2. Сервер получает:
   - домен: santehnik.kz
   - slug: almaty-remont-trub

3. Сервер ищет в БД:
   SELECT * FROM pages 
   WHERE domain = 'santehnik.kz' 
   AND slug = 'almaty-remont-trub'

4. Находит нашу страницу (id=1):
   {
     id: 1,
     slug: 'almaty-remont-trub',
     content: '<html>...</html>',
     domain: 'santehnik.kz' ✅
   }

5. Отправляет контент браузеру
   → Пользователь видит красивую страницу! ✨
```

## ⚙️ Технические Детали

### API Endpoint

```typescript
POST /api/attach-domain

Request:
{
  "domain": "santehnik.kz",
  "pageIds": ["1", "2", "3", "...", "1000"]
}

Response (успех):
{
  "success": true,
  "message": "1000 страниц подвязаны к домену santehnik.kz",
  "attachedCount": 1000,
  "domain": "santehnik.kz"
}

Response (ошибка):
{
  "error": "Неправильный формат домена. Пример: santehnik.kz"
}
```

### Валидация

```
✅ Домен обязателен
✅ Выбрана хоть одна страница
✅ Формат домена правильный (santehnik.kz, не просто santehnik)
✅ Домен в нижнем регистре
```

## ❓ Часто Задаваемые Вопросы

### Q: Можно ли подвязать одни страницы к одному домену, другие - к другому?

**A:** Да! Просто повтори процесс:
1. Выбери другие страницы
2. Укажи другой домен (например: elektrik.kz)
3. Нажми "ПОДВЯЗАТЬ"

Получится:
```
santehnik.kz:      almaty-remont-trub, turgen-remont-trub, ...
elektrik.kz:       almaty-elektrik, turgen-elektrik, ...
remont.kz:         almaty-remont-general, turgen-remont-general, ...
```

### Q: Что если ошибочно подвязал страницы к неправильному домену?

**A:** Нет проблемы! Просто подвязь их к правильному домену снова. Это просто обновит записи в БД.

### Q: Влияет ли это на сам контент страниц?

**A:** Нет! Контент остаётся одинаковым. Меняется только то, на каком домене они видны.

### Q: Можно ли отсоединить страницы от домена?

**A:** Да, но нужна отдельная функция (которую я добавлю позже). Сейчас можешь просто подвязать к другому домену.

### Q: Сколько времени занимает подвязка?

**A:** Очень быстро! Даже 10000 страниц подвязываются за 1-2 секунды.
