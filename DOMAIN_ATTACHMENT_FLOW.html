🌍 Как Домен Показывает Страницы

1. Пользователь Идёт на URL

santehnik.kz/almaty-remont-trub

2. Сервер Получает

домен: santehnik.kz
slug: almaty-remont-trub

3. Запрос в БД

SELECT * FROM pages
WHERE domain = 'santehnik.kz'
AND slug = 'almaty-remont-trub'

4. БД Возвращает

{
  id: 1,
  slug: 'almaty-remont-trub',
  content: '<html>...</html>',
  domain: 'santehnik.kz' ✅
}

5. Браузер Показывает

👁️ Пользователь видит красивую страницу с контентом!
