## 🔄 Как Использовать

### Минимальный Путь (5 минут):

```
1. Зайти в /admin/generate
2. Выбрать Mode 1 (по городам)
3. Заполнить форму (10 городов)
4. Нажать "🚀 Запустить"
5. Получить Batch ID
6. Зайти в /admin/monitor
7. Вставить Batch ID
8. Смотреть прогресс 📊
```
