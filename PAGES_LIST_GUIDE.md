# 📋 Раздел "Все Страницы" - Полный Гайд

## 🎯 Что Это?

**Раздел где ты видишь ВСЕ страницы которые были сгенерированы в системе.**

```
Админ-панель → "Все Страницы" 📋
```

---

## 🖼️ Что Показывается?

### Статистика Сверху:

```
┌──────────┬──────────────┬────────────┬──────────┬────────────┐
│ Всего    │ Опубликовано │ На проверке│ Черновики│ На доменах │
│ 1000     │ 850          │ 100        │ 50       │ 850        │
└──────────┴──────────────┴────────────┴──────────┴────────────┘
```

**Что это показывает:**
- **Всего** - сколько всего страниц сгенерировано
- **Опубликовано** - сколько уже в Google
- **На проверке** - сколько ждут QA
- **Черновики** - сколько не закончены
- **На доменах** - сколько уже подвязаны к доменам

---

## 🔍 Фильтры и Поиск

### Поиск:
```
Поиск по названию или slug...

Примеры:
- "almaty" → найдёт все страницы про Алматы
- "remont" → найдёт все с ремонтом
- "almaty-remont-trub" → точный slug
```

### Фильтр по Статусу:
```
Все статусы ▼
├─ Опубликованы
├─ На проверке
└─ Черновики
```

### Фильтр по Домену:
```
Все домены ▼
├─ santehnik.kz
├─ elektrik.kz
└─ Без домена
```

**Пример:**
```
Ищу: "almaty"
Статус: "Опубликованы"
Домен: "santehnik.kz"

Результат: Все опубликованные страницы про Алматы на santehnik.kz
```

## 📊 Таблица Страниц

### Колонки:

| Колонка | Что Показывает | Пример |
|---------|----------------|--------|
| **Название** | Заголовок страницы | "Ремонт труб в Алматы" |

```
Действия:         👁️ Редактировать 🗑️ Удалить
```

## 🟢 Статусы Страниц

### 1️⃣ Черновик (Draft)
```
Статус: 📝 Черновик
Значит: Страница создана, но не проверена и не опубликована
Что делать: Отправить на QA или отредактировать
```

### 2️⃣ На Проверке (Pending QA)
```
Статус: ⚠️ На проверке
Значит: Страница в очереди на проверку качества
Что делать: Ждите когда проверим, потом либо одобрим либо отправим на доработку
```

### 3️⃣ Опубликована (Published)
```
Статус: ✅ Опубликована
Значит: Страница проверена и опубликована
Что делать: Готова для Google, её можно подвязать к домену
```

## 💾 Полный Цикл Страницы

```
1. ГЕНЕРАЦИЯ
   └─→ Создаёшь 1000 страниц
   └─→ Все становятся "Черновик"

2. QA ПРОВЕРКА
   └─→ Проверяешь вручную
   └─→ Одобряешь → "Опубликована"

3. ПУБЛИКАЦИЯ ВОЛНАМИ
   └─→ Волна 1: публикуешь 100 шт
   └─→ Волна 2: публикуешь ещё 100
   └─→ И так далее...

4. ПОДВЯЗКА К ДОМЕНУ
   └─→ Выбираешь опубликованные страницы
   └─→ Привязываешь к santehnik.kz
   └─→ Домен: "santehnik.kz" ✅

5. ГОТОВО!
   └─→ Страницы видны в интернете
   └─→ Google может их индексировать
```

```

2. Смотрю статистику сверху

Результат:
┌──────────────┐
│ Всего: 1000  │
│ Опубликовано:│ 850 ✅
│ На проверке: │ 100 ⏳
│ Черновики:   │ 50 📝
└──────────────┘

Вывод: 850 готовых, 150 ещё работаю
```

### Пример 2: Найти Все Страницы про Алматы

```
1. Поиск: "almaty"
2. Фильтр статус: "Все статусы"
3. Фильтр домен: "Все домены"

Результат: Показываются только странцы про Алматы
├─ almaty-remont-trub
├─ almaty-ustranenie-zasora
```

```
2. Фильтр домен: "santehnik.kz"

└─ shymkent-remont-trub ✅

Все эти 850 страниц:
- Проверены в QA ✅

```

### Пример 4: Найти Страницы Которые Еще Без Домена

```
1. Фильтр домен: "Без домена"
2. Фильтр статус: "Опубликованы"

Результат: Готовые страницы, но не на домене
├─ city-service-1
├─ city-service-2
└─ [и ещё 48...]

Действие: Выбрать эти 50 и подвязать к domene!
```

## 🎨 Действия со Страницами

### Просмотр (Глаз):
```
Нажимаешь 👁️
→ Открывается предпросмотр страницы
→ Видишь как выглядит
```

### Редактирование (Карандаш):
```
Нажимаешь ✏️
→ Открывается форма редактирования
→ Можешь изменить текст, описание и т.д.
→ Сохраняешь
```

### Удаление (Корзина):
```
Нажимаешь 🗑️
→ Подтверждение "Ты уверен?"
→ Страница удаляется из системы
```

## 📈 Как Использовать Этот Раздел?

### Каждый День:

```
1. Открыть "Все Страницы"
2. Посмотреть статистику:
   - Сколько всего?
   - Сколько опубликовано?
   - Сколько на доменах?

3. Если нужно срочно опубликовать:
   - Фильтр: "Черновики"
   - Отредактировать нужные
   - Отправить на QA

4. Если нужно добавить на домен:
   - Фильтр: "Опубликованы" + "Без домена"
   - Выбрать все 50
   - Перейти в "Подвязать Домен"
   - Привязать к santehnik.kz
```
