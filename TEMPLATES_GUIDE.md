# 📝 Гайд по HTML шаблонам pSEO Platform

## 🎨 3 Готовых Шаблона

У тебя уже есть 3 профессиональных шаблона:

1. **santehnik-template.html** - для сантехнических услуг (синий дизайн)
2. **elektrik-template.html** - для электрических услуг (оранжевый дизайн)
3. **universal-template.html** - универсальный для любой услуги (минималистичный)

Все находятся в папке: `/templates/`

---

## 📋 Переменные (маркеры) в шаблонах

Все шаблоны содержат переменные типа `{{VARIABLE_NAME}}`, которые Claude API будет заполнять контентом.

### 🔑 Основные переменные (в ВСЕ шаблонах):

| Переменная | Описание | Пример |
|-----------|---------|---------|
| `{{CITY}}` | Город | Almaty, Turgen, Shymkent |
| `{{SERVICE}}` | Название услуги | Ремонт труб, Укладка плитки |
| `{{DOMAIN}}` | Название домена | santehnik.kz |
| `{{META_DESCRIPTION}}` | Meta description (160 символов) | Ремонт труб в Алматы... |
| `{{KEYWORDS}}` | Ключевые слова | ремонт труб, сантехник, Алматы |
| `{{FOOTER_PHONE}}` | Телефон | +7-701-123-4567 |
| `{{FOOTER_EMAIL}}` | Email | info@santehnik.kz |
| `{{FOOTER_ADDRESS}}` | Адрес | ул. Абилова, 123, Алматы |

## 🚀 Как всё работает (полный цикл):

### 1️⃣ Загружаешь шаблон в админ-панель

```
Admin → Шаблоны → + Загрузить Шаблон
├─ Выбрать файл: santehnik-template.html
├─ Нишу: "Сантехнические услуги"
├─ Домены: ☑ santehnik.kz
└─ Сохранить
```

### 2️⃣ Генерируешь контент

```
Admin → Генерация → Выбрать режим
├─ Домен: santehnik.kz
├─ Нишу: Сантехнические услуги
├─ Шаблон: santehnik-template.html
├─ Города: Almaty, Turgen, Shymkent
└─ 🚀 Генерировать
```

### 3️⃣ Платформа генерирует

```
Claude API получает промпт:

"Заполни эти переменные для SEO страницы:
Город: Almaty
Услуга: Сантехнические услуги
Домен: santehnik.kz

Переменные:
- {{CITY}}: Almaty
- {{SERVICE}}: Сантехнические услуги
- {{INTRO_CONTENT}}: [генерируй введение 200 слов]
- {{PROBLEM_1_TITLE}}: [генерируй название проблемы]
- {{PROBLEM_1_DESC}}: [генерируй описание проблемы]
[и т.д. для всех переменных]

Возвращай JSON с заполненными переменными"
```

### 4️⃣ Плаформа подставляет в шаблон

```
Берёт santehnik-template.html

Заменяет:
{{CITY}} → Almaty
{{SERVICE}} → Сантехнические услуги
{{INTRO_CONTENT}} → [сгенерированный текст]
{{PROBLEM_1_TITLE}} → [сгенерированный текст]
[и т.д.]

Получает готовый HTML с красивым дизайном и контентом!
```

### 5️⃣ Проверяешь в QA

```
Admin → QA → Показывает страницу
├─ Содержание: ✅ OK
├─ Дизайн: ✅ OK
├─ Уникальность: 82%
└─ [Одобрить] [Отклонить]
```

### 6️⃣ Публикуешь волнами

```
Admin → Публикация → Волна 1
├─ 3 страницы готовы
├─ Опубликовать?
└─ [Опубликовать Сейчас]
```

### 7️⃣ Готово! На домене!

```
santehnik.kz/almaty-santehnik
santehnik.kz/turgen-santehnik
santehnik.kz/shymkent-santehnik

Все с твоим красивым дизайном из шаблона! ✨
```

## 🎯 Если хочешь изменить дизайн:

Можешь редактировать CSS в файлах шаблонов:
- Цвета: измени hex коды (#2563eb на нужный)
- Шрифты: измени font-family
- Отступы: измени padding/margin
- Макет: измени grid-template-columns

**Но не трогай {{ПЕРЕМЕННЫЕ}}** - их заполняет Claude!
