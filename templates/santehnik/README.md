# 📋 Шаблоны для ниши "Сантехнические услуги"

## 📂 Структура папок

```
templates/santehnik/
├── README.md                          ← Ты здесь (навигация)
├── TZ.txt                             ← ТЗ для ГЛАВНОЙ СТРАНИЦЫ
├── sample.html                        ← ГЛАВНАЯ СТРАНИЦА (генерирует /almaty-santehnik/)
│
└── services/                          ← ПАПКА СО ШАБЛОНАМИ ДЛЯ УСЛУГ
    ├── remont-trub/
    │   ├── sample.html                ← Шаблон для "Ремонта труб"
    │   └── TZ.txt                     ← ТЗ для этой услуги
    │
    ├── ustranenie-zasora/
    │   ├── sample.html                ← Шаблон для "Прочистки засоров"
    │   └── TZ.txt
    │
    ├── remont-unitaza/
    │   ├── sample.html                ← Шаблон для "Ремонта унитазов"
    │   └── TZ.txt
    │
    ├── ustanovka-rakoviny/
    │   ├── sample.html                ← Шаблон для "Установки раковин"
    │   └── TZ.txt
    │
    └── remont-vodoonagrevatelya/
        ├── sample.html                ← Шаблон для "Ремонта водонагревателей"
        └── TZ.txt
```

---

## 🎯 Как использовать

### 📖 ШАГ 1: Главная страница

**Файлы:**
- `sample.html` (в корне santehnik/)
- `TZ.txt` (в корне santehnik/)

**Что это:**
- Главная страница ниши → генерирует `/almaty-santehnik/`, `/turgen-santehnik/` и т.д.
- Общая информация по всем сантехническим услугам

**Как редактировать:**
1. Открой `TZ.txt` — читай инструкцию по маркерам
2. Открой `sample.html` — редактируй дизайн
3. Расставляй маркеры согласно ТЗ
4. Сохрани файл

---

### 🔧 ШАГ 2: Шаблоны для услуг

**Файлы:**
- `services/remont-trub/sample.html`
- `services/remont-trub/TZ.txt`
- `services/ustranenie-zasora/sample.html`
- `services/ustranenie-zasora/TZ.txt`
- ... и т.д.

**Что это:**
- Каждая папка = одна услуга
- Шаблон для "Ремонта труб" → генерирует `/almaty-remont-trub/`, `/turgen-remont-trub/` и т.д.
- Шаблон для "Прочистки засоров" → генерирует `/almaty-ustranenie-zasora/`, `/turgen-ustranenie-zasora/` и т.д.

**Как редактировать:**
1. Заходишь в `services/remont-trub/`
2. Открываешь `TZ.txt` — инструкция (маркеры одинаковые, как в главной)
3. Открываешь `sample.html` — редактируешь дизайн ДЛЯ ЭТОЙ УСЛУГИ
4. Расставляешь маркеры
5. Сохраняешь файл

## 📍 Какие маркеры используются везде?

**Одни и те же маркеры:**
- `{{meta.title}}`
- `{{hero.h1}}`
- `{{section.heading}}`
- `{{interlinking.item.label}}`
- ... и т.д.

**Разница только в контенте:**
- Главная → о всех услугах
- Ремонт труб → только о ремонте труб
- Прочистка засоров → только о прочистке засоров
- и т.д.

## 🚀 Загрузка в платформу (потом)

Когда все шаблоны готовы, ты загружаешь их в платформу:

### Главная страница:
```
/admin/templates
→ Загрузить шаблон
→ Выбрать: santehnik/sample.html
→ Тип: main (главная)
→ Сохранить
```

### Шаблоны услуг:
```
/admin/templates
→ Загрузить шаблон
→ Выбрать: services/remont-trub/sample.html
→ Тип: service
→ Услуга: remont-trub
→ Сохранить
```

Платформа проверит все маркеры и либо примет, либо скажет, что не хватает.

## ❓ Часто задаваемые вопросы

**Q: Может ли главная и услуги использовать один и тот же дизайн?**
A: Да, если дизайн универсален. Но лучше сделать разные — для каждой услуги свой.

**Q: Нужно ли заполнять все услуги?**
A: Нет, можно заполнить только те, которые захочешь генерировать. Остальные оставить пустыми.

**Q: Что если дизайн одинаков для всех услуг?**
A: Скопируй `services/remont-trub/sample.html` в остальные папки (Ctrl+C → Ctrl+V).

**Q: Сколько услуг можно добавить?**
A: В конфиге `santehnik.json` указано 5 услуг. Можешь добавить больше, создав новые папки.

**Q: Если я не создам шаблон для услуги, что будет?**
A: Платформа будет использовать главный шаблон (sample.html) для всех услуг.
